# Payoff
 **(Work-in-Progress)**

Manage your debts in one place.

https://payoffweb.vercel.app/

> [!IMPORTANT]  
> This app is not optimized for phones yet.


## Made with
- Next.js
- React
- Pocketbase
- Tailwind CSS
- Framer Motion
